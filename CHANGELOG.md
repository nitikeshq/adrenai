# Changelog

## 1.0.0

- Stable local-first CLI for inspecting repositories and configuring AI coding agents.
- Deterministic workflow recommendations across development, architecture, design,
  security, testing, SEO, mobile, writing, office-document, and marketing categories.
- Native Codex, Claude Code, Copilot, Cursor, Kiro, Gemini, and portable `AGENTS.md`
  outputs.
- Preview-first generation, create-only writes, synchronization, lockfiles, drift
  detection, validation, diagnostics, and allowlisted quality gates.
- Release automation for npm trusted publishing, GitHub Releases, and Homebrew formula
  generation.
