# Security Policy

## Reporting A Vulnerability

Do not open public issues for suspected vulnerabilities. Use GitHub's private
security advisory reporting for this repository and include reproduction steps,
affected versions, and the potential impact.

Maintainers will acknowledge a report within seven days and will coordinate
disclosure after a fix is available.

## Scope

Security reports may cover the AdrenAI CLI, generated configuration, pack
validation, repository inspection, and release artifacts.

AdrenAI treats external packs and agent instructions as untrusted input.
