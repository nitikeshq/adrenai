# AdrenAI Product Hunt Asset Kit

This directory contains production specifications and approved launch copy for
AdrenAI's Product Hunt gallery, logo treatment, screenshots, demo video, and
social posts.

## Launch Positioning

**Product name:** AdrenAI

**Primary tagline:** Give every AI coding agent the right repository context

**Short description:** AdrenAI inspects your repository, recommends compatible
agent guidance, and safely configures Codex, Claude Code, Copilot, Cursor, Kiro,
Gemini, or a portable `AGENTS.md`. Core behavior is local and deterministic.

**Core promise:** One repository-aware configuration layer for every AI coding
agent, without silently overwriting user-authored files.

**Primary audience:** Developers and teams using one or more AI coding agents.

**Primary call to action:** Try AdrenAI on a repository

## Product Hunt Asset Requirements

- Gallery canvas: `1270 x 760 px`
- Gallery images required for visibility: at least 2
- Maximum gallery image or GIF size: under `3 MB`
- Thumbnail: square image
- Video: full public or unlisted YouTube URL; shortened links are unsupported
- GIF first frame must work as a static thumbnail because GIFs animate on hover

Product Hunt specifications can change. Confirm them against Product Hunt's
submission form immediately before upload.

## Production Order

1. Finalize the logo and color tokens.
2. Capture the screenshots in `screenshot-checklist.md`.
3. Produce gallery slides from `gallery-slides.md`.
4. Record the demo using `demo-video-storyboard.md`.
5. Schedule approved copy from `social-posts.md`.

## Copy Rules

- Do not claim AdrenAI uses AI; its current core is local and deterministic.
- Do not claim support for an agent unless its generated output has been tested.
- Do not publish invented customer counts, time savings, token savings, or
  performance metrics.
- Use "safe synchronization" only with the explanation that AdrenAI updates
  unchanged files it owns and blocks user-modified or unmanaged files.
- Use screenshots from a clean demo repository. Never expose secrets, personal
  paths, private repository names, or unrelated terminal history.

## Files

- `gallery-slides.md`: Product Hunt gallery copy and layout specifications
- `logo-usage.md`: logo, thumbnail, spacing, and background rules
- `screenshot-checklist.md`: required product screenshots and capture standards
- `demo-video-storyboard.md`: 60-second demo narrative and shot list
- `social-posts.md`: launch copy variants for social channels and communities
