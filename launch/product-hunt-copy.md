# Product Hunt Copy

## Product name

AdrenAI

## Tagline

Give every AI coding agent the right repository guidance

## Short description

AdrenAI inspects your repository, detects existing AI-agent instructions, and
recommends a small, compatible setup for Codex, Claude Code, Cursor, Copilot,
Kiro, Gemini, and portable `AGENTS.md` workflows.

## Long description

AI coding agents become more useful when they understand a repository's
architecture, conventions, tests, and security requirements. Maintaining that
guidance across several agent-specific files is repetitive and easy to get
wrong.

AdrenAI is an open-source, local-first CLI that helps establish and maintain a
consistent starting point.

It can:

- Inspect repository technologies and existing agent configuration.
- Read common Codex, Claude Code, Cursor, GitHub Copilot, Kiro, Gemini, and
  generic instruction locations.
- Diagnose duplicate, conflicting, broken, or oversized guidance.
- Recommend focused guidance packs using deterministic local rules.
- Preview native configuration for selected agents.
- Create missing generated files without overwriting existing files.
- Detect changes to files managed by AdrenAI.

AdrenAI works without an AI provider. Its built-in catalog also provides
deterministic starting points for design, SEO, mobile, writing, office-document,
and marketing workflows.

## First comment summary

AdrenAI exists to answer a practical question:

> What guidance does this repository actually need, and how do we keep it
> consistent across AI coding agents?

The current release is the first stable open-source version. We are launching
to validate the workflow with real repositories and learn which diagnostics,
adapters, and guidance packs provide lasting value.
