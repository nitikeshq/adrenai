# Honest Positioning

## Positioning statement

For developers using one or more AI coding agents, AdrenAI is an open-source
repository guidance tool that inspects existing configuration, recommends a
minimal shared baseline, and generates native agent files without requiring an
AI provider.

Unlike a large skills marketplace, AdrenAI focuses on selecting and maintaining
a small, compatible set of repository-specific instructions.

## Launch promise

> Understand and improve your repository's AI-agent guidance without starting
> from a blank file.

## What is available now

- Local repository and agent-configuration inspection
- Deterministic recommendations with evidence
- Declarative built-in guidance packs
- Native output adapters for several coding agents
- Preview-first, create-only file generation
- Existing-instruction diagnostics
- Managed-file drift detection
- Deterministic workflow packs for development, design, SEO, mobile, writing,
  office-document, and marketing use cases

## What is not available yet

- Automatic installation or configuration of agent applications
- Safe merging or updating of existing user-authored instructions
- Signed community packs or a hosted marketplace
- Guaranteed compliance or security certification
- Complete support for every language, framework, and agent
- A hosted visual editor for design, writing, spreadsheet, or marketing workflows

## Claims to avoid

Do not claim that AdrenAI:

- Guarantees agents follow instructions.
- Makes generated code secure.
- Replaces testing, review, or engineering judgment.
- Supports every coding agent.
- Automatically fixes repository architecture.
- Saves a specific amount of time or tokens without measured evidence.
- Has official endorsement from supported agent vendors.

## Competitive framing

AdrenAI complements agent-native instruction files and skill systems. It does
not replace Codex, Claude Code, Cursor, Copilot, Kiro, Gemini, or their native
configuration formats.

Its value is the repository-aware layer that inspects, recommends, translates,
and diagnoses guidance across those formats.

## Ideal launch user

- Uses at least one AI coding agent regularly.
- Maintains repository-specific instructions or wants to start.
- Values local-first and deterministic tooling.
- Is willing to review generated guidance.
- Can provide concrete feedback from a real repository.

## Poor-fit launch user

- Expects AdrenAI to build an application autonomously.
- Needs certified policy enforcement.
- Wants a hosted no-code interface.
- Requires stable enterprise support today.
