# Visual Design and Animation Catalog

The reviewed design catalog contains more than 20 strategies in each major
category: visual systems and motion systems. It covers visual direction,
typography, color, backgrounds, layout, responsive behavior, accessibility,
interaction, frameworks, motion, reduced motion, and performance.

Framework strategies cover shadcn/ui composition, Radix UI primitives,
Material components, Bootstrap components, and framework-neutral systems.
Compatibility and conflict metadata supports deliberate hybrid composition.
Every strategy includes enforceable brand-token constraints.

All descriptions and rationales are original and neutral. They do not copy
proprietary skills or imply endorsement by a designer, brand, or framework.
`catalog/workflows/design-delivery.json` provides review and accessibility gates.
