# Software and Mobile Engineering Catalog

`catalog/taxonomies/software-mobile.json` contains reviewed declarative
strategies for two major engineering categories:

- 20 software-engineering strategies covering architecture, methodology, APIs,
  data, testing, security, deployment, observability, and documentation.
- 20 mobile-engineering strategies covering native iOS, native Android, React
  Native, Flutter, accessibility, offline-first behavior, synchronization,
  app-store release, testing, performance, privacy, and operations.

Strategies include a bounded description, reviewed rationale, maturity, risk,
license, prerequisites, conflicts, compatibility, capability, and expected
deliverable. They are suggestions with explicit tradeoffs, not universal
best-practice claims.

`catalog/workflows/mobile-delivery.json` provides a representative approval and
gate-driven mobile delivery process.
